REST API endpoints and project folder structure reference

Auth Service — /api/v1/auth

POST/registerRegister new user
POST/loginLogin, returns JWT
POST/refreshRefresh access token
POST/logoutInvalidate refresh token
GET/oauth2/authorize/googleGoogle OAuth2 redirect
POST/verify-emailVerify email token

User Service — /api/v1/users

GET/meCurrent user profile
PUT/meUpdate profile
GET/{id}Get user by ID (admin)
GET/List all users (admin)
PATCH/{id}/roleChange user role (admin)
DELETE/{id}Delete user (admin)

Queue Service — /api/v1/queues

POST/Create queue for center+date
GET/{id}Get queue details
GET/centers/{centerId}/todayToday's active queue
POST/{id}/joinJoin queue → returns ticket
GET/{id}/position/{ticketNo}Current position + ETA
POST/{id}/nextCall next (staff only)
POST/{id}/skip/{ticketNo}Skip absent user (staff)
GET/ws/{queueId}WebSocket — live updates

Appointment Service — /api/v1/appointments

POST/Book appointment
GET/meMy appointments
GET/{id}Appointment details
PUT/{id}/rescheduleReschedule appointment
DELETE/{id}Cancel appointment
GET/centers/{centerId}Center's appointments (staff)

Notification Service — /api/v1/notifications

GET/meMy notifications
PATCH/{id}/readMark as read
POST/sendManual send (admin)

Analytics Service — /api/v1/analytics

GET/centers/{id}/reportDaily report for center
GET/centers/{id}/wait-timesHistorical wait times
GET/summarySystem-wide dashboard (admin)
